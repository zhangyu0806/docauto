# 企业文档自动化 - 演示案例

## 案例1：发票批量处理

### 场景描述
某财务公司每月收到500+张发票PDF，需要：
1. 提取发票明细到Excel
2. 核对金额、税号
3. 按日期分类整理

### 传统方案
- 人工复制粘贴：8小时/月
- 容易出错：漏看、输错数字
- 效率低下：重复劳动

### 自动化方案
```bash
# 一行命令搞定
python docauto.py pdf-to-excel ./发票文件夹/ -o ./输出/

# 数据清洗
python docauto.py excel-clean 发票数据.xlsx -o 发票数据_cleaned.xlsx
```

### 效果
- 时间：8小时 → 5分钟
- 准确率：95% → 99%
- 成本：0元（开源工具）

## 案例2：合同模板批量生成

### 场景描述
某律所为20家客户起草合同，每份合同需要：
1. 填充客户名称、金额、日期等变量
2. 保持格式统一
3. 快速交付

### 传统方案
- Word查找替换：30分钟/份 × 20 = 10小时
- 容易遗漏：某处忘记替换
- 格式混乱：复制粘贴出错

### 自动化方案

**步骤1：创建模板**
```bash
python docauto.py template --create-sample 合同模板.docx
```

**步骤2：准备数据（Excel）**
| 客户名称 | 合同金额 | 交付日期 | 付款方式 |
|---------|---------|---------|---------|
| XX科技  | 50000   | 2026-03-01 | 分期 |
| YY贸易  | 80000   | 2026-03-15 | 一次性 |

**步骤3：批量生成**
```bash
python docauto.py template 合同模板.docx -d 客户数据.xlsx -o ./生成合同/
```

### 效果
- 时间：10小时 → 5分钟
- 准确率：90% → 100%
- 可复用：下月直接用

```bash
# 用Excel数据透视表快速分析
# （或用pandas进一步处理）
```

### 效果
- 时间：4小时 → 10分钟

## 案例4：财务报表自动生成

### 场景描述
月度财务报表需要：
1. 汇总各分公司数据
2. 生成Word报告
3. 发送给管理层

### 自动化方案

**数据准备**（分公司Excel）：
| 分公司 | 收入 | 成本 | 利润 |
|-------|------|------|------|
| 北京   | 100  | 60   | 40   |
| 上海   | 150  | 90   | 60   |

**模板**（Word）：
```
## {{月份}}财务报表

### 总体情况
本月总收入：{{总收入}}万元
总成本：{{总成本}}万元
净利润：{{净利润}}万元

### 分公司数据
{% for row in 数据 %}
- {{row['分公司']}}：收入{{row['收入']}}万，利润{{row['利润']}}万
{% endfor %}
```

**执行生成**：
```bash
python docauto.py template 财报模板.docx -d 分公司数据.xlsx -o 月度财报.docx
```

### 效果
- 时间：2小时 → 3分钟
- 格式统一、专业
- 每月复用
