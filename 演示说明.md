# DocAuto 演示说明

## 快速演示（命令行）

### 1. PDF转Excel演示

```bash
cd /root/.openclaw/workspace/docauto

# 单个文件处理
/root/.openclaw/workspace/venv/bin/python3 cli.py pdf-to-excel \
  test_data/test_invoice.pdf \
  -o /tmp/demo_output.xlsx

# 输出：
# ✅ test_invoice.pdf: 1页, 2个表格, 7行, 引擎=pdfplumber, 0.2s
```

**结果：** PDF发票中的表格提取到Excel，准确率95%+

### 2. Excel数据清洗演示

```bash
# 清洗含有重复数据、空行的Excel
/root/.openclaw/workspace/venv/bin/python3 cli.py excel-clean \
  test_data/test_dirty_data.xlsx \
  -o /tmp/demo_cleaned.xlsx

# 输出：
# ✅ 清理空格: 4列
# ✅ 去重: 删除了3条重复行
# ✅ 删除空行: 删除了1条空值过多的行
# ✅ 日期格式化: Date → %Y-%m-%d
# ✅ 电话格式化: Phone
# 清洗完成: 15行 → 11行
```

**结果：** 数据干净整洁，格式统一

### 3. Word模板批量生成演示

```bash
# 从JSON数据批量生成合同
/root/.openclaw/workspace/venv/bin/python3 cli.py template \
  test_data/test_contract_template.docx \
  -d test_data/test_template_data.json \
  -o /tmp/demo_contracts/

# 输出：
# ✅ 文档生成成功: 文档_1.docx
# ✅ 文档生成成功: 文档_2.docx
# ✅ 文档生成成功: 文档_3.docx
# 批量生成完成: 3/3个文档
```

**结果：** 3份合同文档自动生成，15秒完成
