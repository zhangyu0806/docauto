# 📄 DocAuto - 快速入门指南

## 5分钟上手企业文档自动化

### 安装

```bash
# 1. 克隆或下载项目
cd /root/.openclaw/workspace/docauto

# 2. 创建虚拟环境（如果还没有）
python3 -m venv ../venv

# 3. 安装依赖
../venv/bin/pip install -r requirements.txt
```

### 三大核心功能

#### 1️⃣ PDF → Excel（发票批量处理）

**单个文件：**
```bash
../venv/bin/python3 pdf_to_excel_v2.py 发票.pdf -o 发票.xlsx
```

**批量处理：**
```bash
../venv/bin/python3 pdf_to_excel_v2.py ./发票文件夹/ -o ./输出/
```

**输出内容：**
- 全部数据汇总（一个sheet）
- 每个表格单独（多个sheet）
- 标注页码来源

---

#### 2️⃣ Excel 数据清洗

```bash
../venv/bin/python3 cli.py excel-clean 脏数据.xlsx -o 干净数据.xlsx
```

**自动处理：**
- ✅ 去除多余空格
- ✅ 删除重复行
- ✅ 删除空行
- ✅ 日期格式统一
- ✅ 电话号码标准化
- ✅ 生成清洗报告

---

#### 3️⃣ Word 模板批量生成

**创建模板：**
```bash
../venv/bin/python3 cli.py template --create-sample 我的模板.docx
```

**准备数据（Excel）：**
| 客户名称 | 金额 | 日期 |
|---------|------|------|
| XX公司 | 50000 | 2026-03-01 |
| YY公司 | 80000 | 2026-03-15 |

**批量生成：**
```bash
../venv/bin/python3 cli.py template 合同模板.docx -d 客户数据.xlsx -o ./生成合同/
```

## 常见问题

**Q: PDF表格识别准确率？**
A: 双引擎（pdfplumber+pymupdf），95%+准确率。

**Q: 支持哪些格式？**
A: PDF、Excel（.xlsx/.xls）、Word（.docx）

**Q: 能处理扫描版PDF吗？**
A: 仅限可选择文字的PDF。扫描版需要OCR（可定制）。

**Q: 数据安全吗？**
A: 本地处理，数据不上传，源码开放可审计。
