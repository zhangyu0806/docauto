## 问题1：如何用Python批量处理PDF文件提取表格？

### 回答

我曾遇到一个实际需求：财务朋友有1000页PDF发票需要提取表格到Excel，手动处理要3天。

用Python写了个脚本，30分钟搞定。

---

## 核心方案

使用 **pdfplumber** 库提取PDF表格：

```python
import pdfplumber
import pandas as pd

def extract_tables(pdf_path):
    all_tables = []
    with pdfplumber.open(pdf_path) as pdf:
        for page in pdf.pages:
            tables = page.extract_tables()
            for table in tables:
                df = pd.DataFrame(table[1:], columns=table[0])
                all_tables.append(df)
    return all_tables
```

**为什么选pdfplumber？**
- 精准识别表格边界
- 保留原始格式
- 支持复杂表格
- 比PyPDF2、tabula-py更稳定

## 2. Word模板：docxtpl

**用途**：基于模板批量生成文档

```python

template = DocxTemplate('合同模板.docx')

```

**适用场景**：
- 批量生成合同
- 批量生成报告
- 批量生成通知函

---

## 3. Excel处理：pandas + openpyxl

**用途**：数据清洗、分析、导出

```python
import pandas as pd

# 去重
df = df.drop_duplicates()

# 标准化日期
df['日期'] = pd.to_datetime(df['日期']).dt.strftime('%Y-%m-%d')

# 导出
df.to_excel('输出.xlsx', index=False)
```
